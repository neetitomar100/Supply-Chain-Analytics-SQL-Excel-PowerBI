# Supply Chain Analytics Project (SQL | Excel | Power BI)

## Project Overview

This project analyzes **Supply Chain data** to identify key insights related to **sales performance, profit trends, and operational efficiency**.

The project demonstrates how raw business data can be transformed into meaningful insights using **SQL for data analysis**, **Excel for pivot-based dashboards**, and **Power BI for interactive visualization**.

The goal of this project is to showcase **data analysis, dashboard creation, and business insight generation skills** using real-world style supply chain data.

---

## Tools & Technologies Used

* **SQL** – Data querying and analysis
* **Excel** – Pivot tables, pivot charts, and dashboard creation
* **Power BI** – Interactive dashboards and data visualization
* **DAX (Data Analysis Expressions)** – Custom measures in Power BI
* **GitHub** – Project documentation and version control

---
